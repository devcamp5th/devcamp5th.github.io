โจทย์รอบคัดเลือก

สร้าง Profile Page มา 1 หน้า ที่แสดงตัวตนของคุณอย่างมีสไตล์ โดยใช้ Bootstrap เป็น Framework หลักในการแสดงผล

นำผลงาน Profile Page ของคุณไปแสดงที่ username.github.io (Github Page) โดย username คือชื่อ Github Username ของคุณ

Github Repository ที่คุณสร้างขึ้นสำหรับ Github Page จะต้องเป็น Private Repository

จากนั้นส่ง URL ของ Github Page ของคุณ ที่ Form ด้านล่างนี้ (ต้อง Login Google Account ก่อนส่ง)

เงื่อนไขการผ่านรอบคัดเลือก

  1. คัดเลือกผู้ผ่านเข้ารอบ 60 คน ที่ทำตามเงื่อนไขข้างต้นได้ถูกต้อง และได้คะแนน 60 อันดับแรกจากคณะกรรมการ
  2. กำหนดส่งภายในวันที่ 21 มิถุนายน 2560
  3. การตัดสินของคณะกรรมการถือเป็นที่สิ้นสุด
